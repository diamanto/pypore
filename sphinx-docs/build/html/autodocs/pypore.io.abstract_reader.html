pypore.io.abstract_reader module¶

class pypore.io.abstract_reader.AbstractReader¶

Bases: object

This is an abstract class showing the methods that subclasses must override.

-Subclasses should implement opening full data files. -Subclasses should implement lazy loading with pypore.io.abstract_reader.get_next_blocks()
__init__¶

Opens a data file, reads relevant parameters, and returns then open file and parameters.

Parameters:filename (StringType) – Filename to open and read parameters.

If there was an error opening the files, params will have ‘error’ key with string description.
